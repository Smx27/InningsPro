  <action>
    Implement `recreateStateFromStream(matchId: Id, rules: MatchRules, events: BallEvent[]): MatchEngineState` in `reducer.ts`.
    The function should:
    1. Initialize a new match state using a 'START_MATCH' action.
    2. Map each `BallEvent` back to its corresponding `MatchEngineAction`.
    3. Iterate through the mapped actions, applying each to the state via `matchReducer`.
    4. Return the final state.
